Duration

Contribution

Overview

프로젝트 목표 기획부터 배포까지 전 과정을 주도한 개인 프로젝트로, React 기반 상태 인터랙션을 구현하고 이미지 리소스 최적화를 통해 모바일 환경에서도 안정적으로 동작하는 UI를 구축하는 것을 목표로 했습니다.

Result Screens

이미지 최적화 및 데이터 분석 지표

WebP 포맷 전환을 통한 리소스 전송량 최적화

모바일 사용자의 데이터 부담을 줄이기 위해 장당 약 1.2MB에 달하던 고용량 PNG 에셋을 분석하여 WebP 포맷으로 일괄 전환했습니다. 그 결과, 이미지 용량을 약 97% 절감(1.2MB → 30~70KB)하여 네트워크 환경에 구애받지 않는 안정적인 로딩 환경을 구축했습니다.

GA4 지표 분석을 통한 사용자 반응 검증

배포 후 실시간 사용자 데이터를 수집하여 서비스의 유효성을 측정했습니다. 12문항의 테스트 구조임에도 불구하고 평균 참여 시간 2분 43초를 기록하며 안정적인 체류 흐름을 확인했으며, 인당 평균 2.01회 조회 지표를 통해 재테스트 및 콘텐츠 공유 가치를 수치로 증명했습니다.
