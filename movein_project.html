무브인 팀 프로젝트

Team Project

Duration

6.5 Weeks

Contribution

50%

Overview

프로젝트 목표 기존 사이트 리뉴얼이 아닌 창작형 팀 프로젝트로, 모바일 환경에서 사용자가 빠르게 콘텐츠를 탐색할 수 있는 가벼운 정보 흐름과 안정적인 UI 구현을 목표로 했습니다.

내 역할

조장

기획·퍼블리싱·협업 운영 총괄

  • 퍼블리싱·UI 구현

    • 모바일 중심 프로젝트 메인 페이지 HTML/CSS/JavaScript 퍼블리싱
    • 기본 인터랙션 구현 및 사용자 흐름 정리

  • 이슈 대응·품질 관리

    • 전체 UI 구조 점검 및 기능·스타일 피드백
    • GitHub 초기 세팅 지원 및 브랜치 전략 리서치·정리

  • 협업 운영

    • 퍼블리싱·디자인·개발 일정 관리 및 팀 일정 조율
    • Google Drive·Notion 기반 협업 환경 구축
    • 회의록 작성 및 작업 가이드 문서 정리

  • 사후 보완·추가 제작

    • 프로젝트 종료 후 서브 페이지 UI 보완 및 스타일 정제
    • 모바일 중심 결과물 보완을 위한 PC 전용 랜딩 페이지 제작

협업 범위 개발/리뷰 과정에서 CSS와 스크립트가 충돌하는 이슈가 발생해 스타일 충돌 구간을 정리하고 수정 방향을 조율했습니다. 프로젝트 종료 후에는 완성도를 높이기 위해 팀원이 제작한 서브 페이지의 CSS를 추가 보완했고, 모바일 중심 결과물을 설명하기 위한 PC 전용 랜딩 페이지도 별도로 제작했습니다.

What I Did & Solved

  • 와이어프레임을 기준으로 Figma 디자인 시안을 정리하고, 구현 단계에서 필요한 컴포넌트 구조를 먼저 정의했습니다.
  • 모바일 환경 기준으로 메인 페이지 마크업·CSS 스타일링을 담당하고, 탭/모달 등 기본 UI 인터랙션을 직접 구현했습니다.
  • 반복되는 레이아웃(헤더·푸터)은 공통 컴포넌트로 분리해 팀 작업 효율을 높였고, (헤더·푸터는 jQuery 기반으로) 유지보수를 고려한 구조로 적용했습니다.
  • 코드 리뷰 과정에서 CSS와 스크립트 충돌 이슈가 발생해 문제 구간을 정리하고 스타일 우선순위/클래스 구조를 조정해 화면 깨짐을 해결했습니다.
  • 프로젝트 종료 후 결과물의 완성도를 높이기 위해 팀원이 제작한 서브 페이지 CSS를 추가 보완했으며, 전/후 비교 이미지를 통해 변경 내용을 정리했습니다.
  • 모바일 중심 결과물을 보완 설명하기 위해 PC 전용 랜딩 페이지를 추가 제작해 프로젝트 소개/진입 흐름을 정리했습니다.
  • 일정 관리, 회의록 작성, Notion 문서화(공유 링크/자료 정리), GitHub 연동 가이드 작성 등으로 협업 환경을 정리하고 팀 진행을 지원했습니다.
