종합유기견보호센터 사이트 리뉴얼

Persona Project

Duration

5 Weeks

Contribution

100%

Overview

문제인식 : 기존 종합유기견보호센터 홈페이지는 메인 화면에 다양한 기능과 정보를 한 번에 배치하면서 핵심 기능이 시각적으로 묻히고 사용자 행동 동선에 혼선을 주는 구조 를 가지고 있었습니다. 또한 노후화된 디자인과 일관성 없는 UI 구성으로 인해 종합유기견보호센터의 정체성과 신뢰 이미지가 충분히 전달되지 않는 한계 가 있었습니다.

개선방향 : 정보의 우선순위를 재정립하고, 불필요하게 분산되어 있던 콘텐츠를 정리하여 카드형 UI와 필수 정보 중심 구조로 재배치 했습니다. PC 환경에서는 시각적 밀도를 조절하여 사이트 아이덴티티를 강조하는 브랜드 중심 레이아웃으로 설계하였고, 모바일에서는 사용 목적에 집중해 기능 실행 중심의 직관적인 UI 구조로 재구성하였습니다. 이를 통해 PC와 Mobile을 단순 축소 구조가 아닌, 각 환경의 쓰임에 맞는 분리 설계 전략으로 개선했습니다.

Result Screens

결과물 스크린샷 및 설명

Main Screen

헤더/고정 퀵메뉴/메인 콘텐츠/푸터로 구성된 랜딩 페이지로, grid 및 flex를 활용해 기초 레이아웃을 설계했으며, 슬라이드와 탭 UI로 주요 정보를 한 화면에서 탐색할 수 있도록 설계했습니다. 또한 PC 환경에서의 hover 스타일이 모바일 환경과 충돌하지 않도록 @media (hover: hover)구문을 적용했습니다.

Sub Screen

헤더/고정 퀵메뉴/푸터 공통 레이아웃을 재사용 했으며 상단 + 아래 2컬럼(부가정보/본문) 구조로 구성한 게시글 상세 서브페이지입니다.

Project Point

프로젝트 마무리 후 WAVE 웹사이트를 활용해 자동 웹 접근성 검사를 진행하고, 마크업 구조와 대체 텍스트 중심으로 접근성 요소를 점검했습니다. 수동 검사가 필요한 유튜브 영역을 제외해 접근성 검사를 통과했습니다.
